#### **1. Kubernetes (Current Setup)**
```bash

sudo supervisorctl restart frontend

sudo supervisorctl status
```

```bash

yarn build

# Deploy build folder to hosting platform
```

```dockerfile

WORKDIR /app
COPY frontend/package*.json ./

COPY frontend/ .

EXPOSE 3000
CMD ["yarn", "start"]
```
